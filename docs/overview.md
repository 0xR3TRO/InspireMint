# InspireMint — Overview

## What is InspireMint?

InspireMint is a modern, interactive **motivational quote generator** built with pure HTML, CSS, and JavaScript — no frameworks required. It presents users with inspiring quotes, allows personalization through categories, tags, and keyword search, and features a sleek dark/light theme toggle.

## Goals

- Deliver a fast, lightweight motivational experience in the browser.
- Let users personalize which quotes they see via categories and tags.
- Provide a beautiful, accessible, mobile-first UI with smooth animations.
- Remain fully static — no server or build step needed.

## Key Features

| Feature            | Description                                                          |
| ------------------ | -------------------------------------------------------------------- |
| Random Quotes      | Generate a new motivational quote with one click                     |
| Category Selection | Filter quotes by category: general, success, mindfulness, creativity |
| Tag Filtering      | Narrow results by tags like _discipline_, _growth_, _courage_        |
| Keyword Search     | Free-text search across quote text, author, and tags                 |
| Dark / Light Mode  | Toggle theme with one click; preference saved in `localStorage`      |
| Responsive Layout  | Mobile-first design scales beautifully to desktop                    |
| Smooth Animations  | Fade-in/fade-out transitions when quotes change                      |

## Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, flexbox, animations, `@media` queries
- **JavaScript (ES Modules)** — Modular architecture, `fetch` API for JSON data
